# Bethacon - Ethereum Market Intelligence System

Bethacon is an artificial intelligence system for Ethereum trading. It combines historical price data and technical indicators to generate BUY, SELL, or HOLD market signals, allowing you to test machine learning strategies and trading simulations in a modular and practical way.

## Key Features

- Analysis of historical Ethereum data with calculation of technical indicators (RSI, MACD, momentum, moving averages, volume)
- Machine learning models for market signal classification
- Trading simulator to evaluate signal performance in realistic scenarios
- Modular structure to experiment with different feature engineering techniques and algorithms

## Goal

Provide an educational and practical framework to explore machine learning applied to the cryptocurrency market, with a focus on dataset design, model evaluation, and trading simulation.

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Matplotlib
- Altair

## Getting Started

1. Clone the repository
2. Install dependencies
3. Run the simulator to view signals and test trading strategies
